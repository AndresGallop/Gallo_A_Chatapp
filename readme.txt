## Chat application

This is a chat application built in JavaScript by using the library Socket.io and Vue.js,
to help make the app more dynamic and easy to be built.

The app is a live chat with a astronaut theme, you will be able to connect to the chat
with your own name and chat with other person in the server. It uses socket.io to get
and send the data of the name, messages and the ID of each user.The app runs with express
and a browser synconization.


## TOOLS

CSS3 - SASS
JS - Vue.JS
SOCKET.io
EXPRESS
